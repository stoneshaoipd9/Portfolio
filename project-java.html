People Database


(JAVA Project #1)

Description

The "People Database" project can manage the people list through connecting the database.
In the Frame, there are two text fields that I can enter name and age, one buttons that I can add person, and another button that I can list the people in the list area. Also in the list area, I can delete person by using right mouse click.
In the database file, I have list(all or by id), add, update, and delete classes to execute different functions corresponding to different actions in the frame.

List of technologies

  • MySQL
  • JFrame

Mini CRM


(JAVA Project #2)

Description

The "Mini CRM" project can manage the customer and the rep list through connecting the database.
In the Frame, there are four windows in total. Two of them are add customer and add rep, one of them is the customer and the rep window, the last one is the interaction (between customer and rep) window. By choosing customer from the customer list area and rep from the rep list area, and click the "M-C Interactions" button, you can go to the the interaction window that you can add some note.

List of technologies

  • MySQL
  • JFrame
  • Multi-Windows
